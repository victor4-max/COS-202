## Overview

This repository contains two Python programs developed for the COS 202 course. The projects demonstrate the use of Python to solve basic mathematical and academic calculations using simple programming techniques.

---

## Included Programs

- **`calculator.py`** – Performs addition, subtraction, multiplication, division, modulus, and power operations.
- **`cgpa_calculator.py`** – Calculates CGPA from course scores and credit units, then displays the appropriate degree classification.

---

## Features

- ✓ Menu-driven calculator
- ✓ Input validation and error handling
- ✓ Automatic grade and grade point calculation
- ✓ CGPA computation
- ✓ Degree classification display
